# PyStrukts

A Python library for data structures. This library provides more than 20 advanced data structures not available in the Python standard library, including:

- [ ] Singly Linked List
- [ ] Doubly Linked List
- [ ] Circular Linked List (Single, Double)
- [ ] Binary Search Tree
- [ ] Ternary Search Tree
- [ ] Suffix Tree
- [ ] AVL Tree
- [ ] Red-Black Tree
- [ ] Splay Tree
- [ ] B-Tree
- [ ] B+ Tree
- [ ] B* Tree
- [ ] R-Tree
- [ ] Quadtree
- [ ] Octree
- [ ] Interval Tree
- [ ] Segment Tree
- [ ] Fenwick Tree
- [ ] Disjoint Set
- [ ] Skip List
- [ ] Trie
- [x] Max Heap
- [ ] Graph (Adjacency List)
- [ ] Graph (Adjacency Matrix)
- [ ] Graph (Edge List)
